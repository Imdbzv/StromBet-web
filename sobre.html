Sobre StromBet

StromBet es un bot de Discord creado para brindar diversión con un sistema de economía virtual centrado en juegos de azar, estrategia y entretenimiento para toda una comunidad. Aquí no solo apuestas y ganas Diamantes 💎, sino que también puedes convertirlos en Robux 🪙 si eres hábil con tus apuestas.

¿Para qué sirve?

Sirve para simular un casino dentro de Discord con comandos interactivos. Puedes jugar a juegos como Crash, Limbo, Blackjack, Ruleta, entre otros. Además, puedes ganar recompensas diarias, transferir diamantes a otros usuarios y subir en el ranking.

¿Quién lo creó?
Este bot fue creado por Imdbzv, con el objetivo de ofrecer un bot completo en español para comunidades que buscan minijuegos dentro de su servidor.

¿Qué tecnologías usa?
  • Python con discord.py
  • Base de datos SQLite
  • Flask para mantenerlo activo
  • HTML + GitHub Pages para esta página web

El bot se encuentra en constante desarrollo, así que se seguirán agregando más minijuegos, comandos nuevos y mejoras visuales.
